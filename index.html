Aula de Web 1

Neste primeiro encontro conversamos sobre conceito de web, http,html e uso do git

  • git init - cria um repositório git
  • git status - verificar o estado atual dos arquivos
  • git add [file or .] - adiciona um ou vários arquivos
  • git commit -m - comita os arquivos e adiciona nomes a eles
  • git config user.name "name" - colocar o seu nome
  • git config user.email "email@name" - colocar o seu email
  • git log - ver o histórico de commits
